# 主题与色板

本文记录 `bjcp-arena` 当前阶段的系统色板。主题来源于 Traditional Brewing Challenge LOGO：深棕木质底、金色麦穗和酒花、奶油色高光。整体气质偏传统精酿啤酒与英式酒吧，但裁判端需要长时间阅读和输入，因此默认采用克制功能型方案。

## 色板

| Token           | 色值      | 用途                            |
| --------------- | --------- | ------------------------------- |
| `mahogany`      | `#2a1208` | 深色品牌背景、顶部栏底色        |
| `deepAle`       | `#3b1a0b` | 深棕层次、品牌渐变              |
| `copperPrimary` | `#7a3f12` | 主按钮、交互主色、重点状态      |
| `goldAccent`    | `#e8b94d` | LOGO 呼应、品牌强调、焦点外描边 |
| `cream`         | `#fff8e8` | 深色背景上的文字和高光          |
| `parchment`     | `#f7efe3` | 页面背景                        |
| `paper`         | `#fffaf2` | 表单、列表、信息块背景          |
| `ink`           | `#23180f` | 主文字                          |
| `slate`         | `#495057` | 次级文字                        |
| `hopSuccess`    | `#4f6f32` | 成功、可评分、通过状态          |
| `rubyDanger`    | `#a8322a` | 退出、错误、危险操作            |

## 使用规则

- judge 端先落地该色板；admin 后续如需统一主题，再按同一 token 映射到 antd theme。
- 主操作使用 `copperPrimary`，保证白字对比；`goldAccent` 只用于品牌、强调和焦点，不用于大面积正文背景。
- 页面和表单保持 `parchment` / `paper` 浅底，避免深色酒吧风影响评分输入可读性。
- 状态色不要全部替换成金棕色：成功继续使用酒花绿，危险继续使用红色，以保证裁判端快速识别。
- 代码中优先使用 `--bjcp-*` 和 `--adm-*` CSS 变量；不要把同一颜色值散落到页面组件里。
